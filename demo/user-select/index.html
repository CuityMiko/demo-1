user-select研究

起因是昨天(2017-03-23)有个小伙伴(@灿灿)在一个页面无意中选中了一段文字, 但这个文字又隐藏了, 导致出现了一些诡异的问题, so研究下几种情况下针对不同内容的user-select, pointer-events效果

case1. 期望用户去长按

比如一些图片, 长按有浏览器下载浮层

默认
user-select: none
pointer-events: none
