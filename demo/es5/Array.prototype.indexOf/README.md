# Array.prototype.indexOf

```js
/**
 * @file 查找数组里值的索引
 *
 * @description 注意的是值的比较是以 === 比较的
 * @param {*} val 目标值
 * @param {number|undefined} startIndex 如果为数字则以当前值查找，但结果还是以下标为0开始的索引
 * @return {number} 目标值在数组里的位置，未找到则为-1
 * @example
 *     Array.prototype.indexOf(val [,startIndex]);
 */
```
